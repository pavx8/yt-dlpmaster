# yt-dlpMaster

Десктопный GUI для `yt-dlp`, написанный на Python и PySide6.

English version: [README.md](README.md)

## Возможности
- Ввод URL видео
- Выбор папки сохранения
- Пресеты формата загрузки
- Поддержка прокси (`socks4`, `socks5`, `socks5h`, `http`, `https`)
- Поддержка cookies (профиль браузера или файл cookies.txt)
- Выбор языка интерфейса
- Опциональное перекодирование для совместимости (H.264/AAC для видео, MP3 для аудио)
- Прогресс загрузки и лог статуса
- Переключение между встроенными и системными компонентами во время работы

## Требования
- Python `>=3.10,<3.15`
- Poetry
- `ffmpeg` в `PATH` или комплектные бинарники в проекте

## Быстрый старт
```bash
poetry install
poetry run yt-dlpmaster
```

Запуск с явным языком интерфейса:
```bash
poetry run yt-dlpmaster --lang ru_RU
```

## Локализация
- Исходники переводов находятся в `app/i18n/*.ts`.
- Бинарные файлы переводов - `app/i18n/*.qm`.
- При старте приложения `.qm` собираются автоматически, если отсутствуют или старее `.ts`.

Ручные команды:
```bash
./scripts/build_translations.sh
./scripts/update_translations.sh
```

## Структура проекта
- `app/cli.py` - точка входа CLI (`yt-dlpmaster`)
- `app/ui/` - слой Qt-интерфейса
- `app/core/` - загрузка, ffmpeg, настройки, CA-хелперы
- `app/i18n/` - локализации (`.ts` / `.qm`)
- `scripts/` - утилиты для работы с переводами

## Примечания
- Настройки хранятся в портативном режиме в `settings.ini` в корне проекта.
- При включенной опции приложение предпочитает встроенные бинарники/библиотеки.
- Поиск `ffmpeg` сначала проверяет локальные комплектные пути, затем системный `PATH`.
